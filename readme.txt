== Installation ==

<p>Installation of <strong>Team Showcase and Slider - Team Members Builder</strong> is very simple.</p> 
<blockquote>


<strong>Option 1</strong>

<li> Download the plugin team-showcase-ultimate.zip from Wordpress Plugin Directory.</li>
<li> Unzip file into team-showcase-ultimate folder.</li>
<li> Drop the sb-image-hover-effects plugin folder into your wp-content/plugins folder.</li>
<li> Refresh your Wordpress Administration panels, click on Plugins from the menu.</li>
<li> You will see Team Showcase and Slider - Team Members Builder plugin under Inactive plug-in tab.</li>
<li> To turn on Team Showcase and Slider - Team Members Builder , click  on activate.</li>

</blockquote>

<blockquote>


<strong>Option 2</strong>

<li> Download the plugin team-showcase-ultimate.zip from Wordpress Plugin Directory.</li>
<li> click on add new  under plugins menu.</li>
<li> Upload the available Team Showcase and Slider - Team Members Builder Plugin file and click install now</li>
<li> After installed click on active to tern on.</li>

</blockquote>
